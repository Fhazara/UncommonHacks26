# ExampleProject — Setup Guide

Welcome to ExampleProject! Follow these steps to get started:

1. Install dependencies: `npm install`
2. Configure environment: Check the `.env` file for required variables
3. Run the setup script: `./scripts/setup.sh`
4. Start the development server: `npm run dev`

## Deployment

To deploy to production, follow the standard deployment checklist in `docs/DEPLOYMENT.md`.
